🔵Angular - angular is a frontend framework use to build (SPA) single page application
used for:
1> Dynamic websites
2> develop Scalable application and robust application.

➡️(SPA) single page application - means its change the content dynamically we don't need to reload the page again and again
In SPA only one HTML page load, then change the content by using routing, page is not reload 
Benefits of SPA
Fast navigation
Better user experience
App-like feel

➡️Angular is framework
Framework - framework is set of tools, rules and structure used to build application.
Library gives us only some feautres, framework provide complete package
Routing built-in
Forms built-in
HTTP built-in
Dependency Injection built-in
Testing tools built-in

latest angular version - 21(launched in nov 2025)
launch angular - september 2016
angular js - with javascript
angular - launch typescript
typescript - superset of javascript
package.json → dependencies & scripts
angular.json → project config
src/main.ts → entry point
app.module.ts → root module
app.component.ts/html/css → main component

🔵Angular Architecture:- combination of multiple building blocks
➡️component - basic building block of angular UI
➡️module - group of components and service
➡️service - used for business logic and api calls
➡️template - used for where write html code.
➡️Directive - control dom Behavior(show hide)
➡️Routing - navigation between pages
