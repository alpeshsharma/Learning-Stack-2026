1. What is JavaScript?
JavaScript is a high-level, interpreted scripting language used for web development.

// 2. Difference between var, let, const?
// var -> function scoped
// let -> block scoped
// const -> block scoped and cannot be reassigned

// 3. What is hoisting?

// 4. Difference between == and ===
// == checks value
// === checks value + datatype

5 === "5"  // false

// 6. What is closure?
function outer() {
  let count = 0;

  return function inner() {
    count++;
    return count;
  };
}

// 7. What is lexical scope?
// Inner function can access outer function variables.

// 20. What is debounce?
function debounce(fn, delay) {
  let timer;

  return function() {
    clearTimeout(timer);
    timer = setTimeout(fn, delay);
  };
}

// Throttle -> limits frequency

29. null vs undefined?
Undefined: means a variable is declared but value is not assigned.

console.log(name);       // undefined

null means intentionally empty value.

console.log(user);    // null

30. What is NaN?

34. Rest operator

35. Rest vs Spread?

36. Destructuring

37. Template literals
Template literals are a modern way to create strings in JavaScript using backticks instead of quotes.
`Hello ${name}`

1. Variable Interpolation
const name = "Aman";
const age = 25
console.log(`My name is ${name} and I am ${age} years old.`);

2. Expressions Inside String
const a = 10;
const b = 20;
console.log(`Sum is ${a + b}`);

3. Multi-line Strings
Without template literals:
const text = "Hello\nWorld";

With template literals:
const text = `Hello
World`;
